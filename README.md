## Project Title
Distributed Text Analysis of *The Extermination of the American Bison* using Hadoop HDFS, MapReduce, and MongoDB.

## Project Objective
To store, process, and analyse a large unstructured text dataset using Hadoop HDFS, Hadoop MapReduce, and MongoDB, then evaluate the performance of distributed and NoSQL data storage technologies.

## Dataset
- Dataset: eBook #17748 – *The Extermination of the American Bison*
- Source: Project Gutenberg (provided by the lecturer for the Repeat CA)
- Data Type: Unstructured text (.txt)

## Technologies

- Hadoop MapReduce
- MongoDB
- GitHub
